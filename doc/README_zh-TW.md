# TransPop 🚀
> **讓翻譯再次簡單** ✨

厭倦了 `Cmd+C` -> 打開瀏覽器 -> 輸入 "Google Translate" -> `Cmd+V` -> 哭泣 -> 重複的循環嗎？

是的，我們也一樣。這就是我們構建 **TransPop** 的原因。它就像你菜單欄裡的巴別魚，但沒那麼粘滑。

## 為什麼選擇 TransPop? 🧐

因為生命太短暫，不應該浪費在手動複製粘貼文本到瀏覽器標籤頁上。

### 🌟 讓你驚呼 "哇" 的功能

*   **"雙擊" 魔法**: 按兩次 `Cmd+C` (雙重複製)。嘭！翻譯彈出。就像召喚精靈一樣，只不過是語言精靈。 🧞‍♂️
*   **迷你彈出模式**: 窗口出現在*你光標所在的位置*。我們稱之為 "忍者模式"。你甚至不需要移動鼠標。 🥷
*   **可擴展 UI**: 需要更多空間？點擊迷你界面中的擴展按鈕（箭頭圖標）切換到全尺寸窗口。
*   **托盤圖標**: 我們住在你的狀態欄裡。時刻注視。時刻等待。（以一種非令人毛骨悚然的方式）。 👀
*   **多重提供商**: 
    *   **Google 翻譯 (免費)**: 開箱即用。無需設置。
    *   **OpenAI / Ollama**: 連接到你的本地 LLM (通過 Ollama) 或 OpenAI 兼容 API，獲得更智能的翻譯。
*   **語言交換**: 一鍵反轉流向。`英語 -> 中文` 變成 `中文 -> 英語`。腦洞大開。 🤯
*   **智能關閉**: 選擇關閉窗口時是最小化到托盤還是退出應用程序。你甚至可以告訴它 "不再詢問"。
*   **深色模式**: 因為我們是開發者，淺色模式會灼傷我們的視網膜。 😎

## 🛠 技術棧 (極客內容)

使用純粹的、未摻雜的 **Swift** 和 **SwiftUI** 構建。沒有 Electron。沒有 Chrome 實例吃掉你的內存。只有純粹的原生性能。 🍏

*   **SwiftUI**: 在 macOS 上看起來很棒的聲明式 UI。
*   **AppKit**: 用於處理繁瑣的窗口管理和狀態欄魔法。
*   **Combine**: 用於響應式狀態管理。

## 📥 如何安裝

不想從源碼構建？沒問題。

1.  前往 [Releases](https://github.com/ufolux/TransPop/releases) 頁面。
2.  下載最新的 `.zip` 文件。
3.  解壓並將 `TransPop.app` 拖入你的 `/Applications` 文件夾。

### ⚠️ "無法打開應用，因為無法驗證開發者"?

如果 macOS 抱怨應用已損壞或無法打開（因為我們還沒給 Apple 交 $99/年的保護費），請在終端運行此命令：

```bash
xattr -dr com.apple.quarantine /Applications/TransPop.app
```

然後再次嘗試打開。

## 🏃‍♂️ 如何運行 (開發者)

想在本地運行這個壞小子？給你：


## ⚙️ 配置

通過全視圖中的齒輪圖標訪問 **設置**。

### 通用
*   **語言**: 更改應用程序界面語言。
*   **主題**: 在淺色、深色或系統主題之間切換。
*   **關閉操作**: 選擇關閉窗口時發生的情況（詢問、最小化或退出）。

### 翻譯 API
*   **提供商**: 在 "Google (免費)" 和 "OpenAI 兼容" 之間切換。
*   **OpenAI 兼容設置**:
    *   **API URL**: 默認為 `http://127.0.0.1:11434/v1/chat/completions` (完美適配 Ollama)。
    *   **API Key**: 本地 LLM 可選。
    *   **模型**: 指定模型名稱 (例如 `llama3`, `gpt-4o`, `zongwei/gemma3-translator:1b` 我試過這個，完美工作)。
