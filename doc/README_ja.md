# TransPop 🚀
> **翻訳を再びシンプルに** ✨

**現在は macOS のみサポートしています。他のプラットフォームのサポートは後ほど追加予定です**

<img width="562" height="712" alt="image" src="https://github.com/user-attachments/assets/c6787432-79fd-4f2f-8908-926065c8289c" />

`Cmd+C` -> ブラウザを開く -> "Google 翻訳" と入力 -> `Cmd+V` -> 泣く -> 繰り返す... そんなサイクルにうんざりしていませんか？

あるいは、数語を翻訳するためだけに 300 MB もダウンロードすることに？

ええ、私たちもです。だから **TransPop** を作りました。メニューバーにあるバベルの魚のようなものですが、ぬるぬるはしていません。

## なぜ TransPop なのか？ 🧐

人生は短すぎて、ブラウザのタブにテキストを手動でコピー＆ペーストしている暇はないからです。

### 🌟 "ワオ" と言わせる機能

*   **"ダブルタップ" の魔法**: `Cmd+C` を2回押します（ダブルコピー）。ドーン！翻訳がポップアップします。魔人を呼び出すようなものですが、言語の魔人です。 🧞‍♂️

    <img width="50%" height="50%" alt="image" src="https://github.com/user-attachments/assets/da48497e-7d73-43cc-a3e2-40d9368688d2" />

*   **ミニポップアップモード**: ウィンドウは*カーソルのある場所に*表示されます。私たちはこれを "忍者モード" と呼んでいます。マウスを動かす必要さえありません。 🥷
*   **拡張可能な UI**: もっとスペースが必要ですか？ミニUIの拡張ボタン（矢印アイコン）をクリックして、フルサイズウィンドウに切り替えます。
*   **トレイアイコン**: 私たちはあなたのステータスバーに住んでいます。常に見守り、常に待っています。（不気味な意味ではありません）。 👀
*   **複数のプロバイダー**: 
    *   **Google 翻訳 (無料)**: すぐに使えます。設定不要。
    *   **OpenAI / Ollama**: ローカルLLM (Ollama経由) または OpenAI 互換 API に接続して、よりスマートな翻訳を実現します。
*   **言語スワップ**: ワンクリックで流れを反転。`英語 -> 中国語` が `中国語 -> 英語` になります。衝撃的。 🤯
*   **スマートクローズ**: ウィンドウを閉じるときに、トレイに最小化するかアプリを終了するかを選択します。「二度と聞かない」ように指示することもできます。
*   **ダークモード**: 私たちは開発者であり、ライトモードは網膜を焼くからです。 😎
*   **超軽量**: アプリはわずか 1.8 MB です。数語を翻訳するためだけに 300 MB の Electron アプリをダウンロードする必要はありません。

## 🛠 技術スタック (オタク向け)

純粋で混じりけのない **Swift** と **SwiftUI** で構築されています。Electron はありません。RAM を食う Chrome インスタンスもありません。純粋なネイティブパフォーマンスのみ。 🍏

*   **SwiftUI**: macOS で見栄えの良い宣言型 UI。
*   **AppKit**: 面倒なウィンドウ管理とステータスバーの魔法のために。
*   **Combine**: リアクティブな状態管理のために。

## 📥 インストール方法

ソースからビルドしたくないですか？お任せください。

1.  [Releases](https://github.com/ufolux/TransPop/releases) ページにアクセスします。
2.  最新の `.zip` ファイルをダウンロードします。
3.  解凍して `TransPop.app` を `/Applications` フォルダにドラッグします。

### ⚠️ "開発元を検証できないため開けません"？

macOS がアプリが破損している、または開けないと文句を言う場合（Apple に年間99ドルをまだ支払っていないため）、ターミナルで次のコマンドを実行してください：

```bash
xattr -dr com.apple.quarantine /Applications/TransPop.app
```

その後、もう一度開いてみてください。

## 🏃‍♂️ 実行方法 (開発者向け)

ローカルで実行したいですか？どうぞ：

## ⚙️ 設定

フルビューの歯車アイコンから **設定** にアクセスします。

### 一般
*   **言語**: アプリのインターフェース言語を変更します。
*   **テーマ**: ライト、ダーク、またはシステムテーマを切り替えます。
*   **終了時の動作**: ウィンドウを閉じるときの動作を選択します（毎回確認、トレイに最小化、終了）。

### 翻訳 API
*   **プロバイダー**: "Google (無料)" と "OpenAI 互換" を切り替えます。
*   **OpenAI 互換設定**:
    *   **API URL**: デフォルトは `http://127.0.0.1:11434/v1/chat/completions` (Ollama に最適)。
    *   **API Key**: ローカル LLM の場合はオプション。
    *   **モデル**: モデル名を指定します (例: `llama3`, `gpt-4o`, `zongwei/gemma3-translator:1b` 私はこれを試しましたが、完璧に動作します)。
