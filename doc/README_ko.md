# TransPop 🚀
> **번역을 다시 간단하게** ✨

<img width="562" height="712" alt="image" src="https://github.com/user-attachments/assets/c6787432-79fd-4f2f-8908-926065c8289c" />

`Cmd+C` -> 브라우저 열기 -> "Google 번역" 입력 -> `Cmd+V` -> 좌절 -> 반복... 이 사이클에 지치셨나요?

네, 우리도 그렇습니다. 그래서 **TransPop**을 만들었습니다. 메뉴 막대에 있는 바벨 피쉬 같지만, 덜 끈적거립니다.

## 왜 TransPop인가요? 🧐

인생은 브라우저 탭에 텍스트를 수동으로 복사-붙여넣기 하기에는 너무 짧기 때문입니다.

### 🌟 "와우" 하게 만드는 기능들

*   **"더블 탭" 매직**: `Cmd+C`를 두 번 누르세요 (더블 복사). 펑! 번역이 팝업됩니다. 지니를 소환하는 것 같지만, 언어의 지니입니다. 🧞‍♂️
*   **미니 팝업 모드**: 창이 *커서가 있는 바로 그곳에* 나타납니다. 우리는 이것을 "닌자 모드"라고 부릅니다. 마우스를 움직일 필요조차 없습니다. 🥷
*   **확장 가능한 UI**: 더 많은 공간이 필요하신가요? 미니 UI의 확장 버튼(화살표 아이콘)을 클릭하여 전체 크기 창으로 전환하세요.
*   **트레이 아이콘**: 우리는 당신의 상태 표시줄에 살고 있습니다. 항상 지켜보고, 항상 기다립니다. (소름 끼치는 방식은 아닙니다). 👀
*   **다중 공급자**: 
    *   **Google 번역 (무료)**: 즉시 사용 가능합니다. 설정이 필요 없습니다.
    *   **OpenAI / Ollama**: 로컬 LLM (Ollama 경유) 또는 OpenAI 호환 API에 연결하여 더 스마트한 번역을 받으세요.
*   **언어 전환**: 클릭 한 번으로 흐름을 반전시킵니다. `영어 -> 중국어`가 `중국어 -> 영어`가 됩니다. 놀랍죠. 🤯
*   **스마트 닫기**: 창을 닫을 때 트레이로 최소화할지 앱을 종료할지 선택하세요. "다시 묻지 않음"으로 설정할 수도 있습니다.
*   **다크 모드**: 우리는 개발자이고 라이트 모드는 망막을 태우기 때문입니다. 😎

## 🛠 기술 스택 (너드들을 위한)

순수하고 섞이지 않은 **Swift**와 **SwiftUI**로 구축되었습니다. Electron은 없습니다. RAM을 잡아먹는 Chrome 인스턴스도 없습니다. 오직 순수한 네이티브 성능뿐입니다. 🍏

*   **SwiftUI**: macOS에서 보기 좋은 선언형 UI.
*   **AppKit**: 까다로운 창 관리와 상태 표시줄 마법을 위해.
*   **Combine**: 반응형 상태 관리를 위해.

## 📥 설치 방법

소스에서 빌드하고 싶지 않으신가요? 저희가 해결해 드립니다.

1.  [Releases](https://github.com/ufolux/TransPop/releases) 페이지로 이동합니다.
2.  최신 `.zip` 파일을 다운로드합니다.
3.  압축을 풀고 `TransPop.app`을 `/Applications` 폴더로 드래그합니다.

### ⚠️ "개발자를 확인할 수 없기 때문에 앱을 열 수 없습니다"?

macOS가 앱이 손상되었거나 열 수 없다고 불평하는 경우 (우리가 아직 Apple에 연간 $99를 지불하지 않았기 때문입니다), 터미널에서 다음 명령을 실행하세요:

```bash
xattr -dr com.apple.quarantine /Applications/TransPop.app
```

그런 다음 다시 열어보세요.

## 🏃‍♂️ 실행 방법 (개발자용)

로컬에서 실행해 보고 싶으신가요? 여기 있습니다:

```bash
# 1. 리포지토리 클론 (당연하죠)
git clone https://github.com/ufolux/TransPop.git

# 2. macos 폴더로 이동
cd macos

# 3. 실행! 🚀
swift run
```

## 📦 빌드

릴리스 버전을 빌드하고 싶으신가요?

```bash
cd macos
swift build -c release
```

## ⚙️ 설정

전체 보기의 기어 아이콘을 통해 **설정**에 액세스하세요.

### 일반
*   **언어**: 앱 인터페이스 언어를 변경합니다.
*   **테마**: 라이트, 다크 또는 시스템 테마 간에 전환합니다.
*   **닫기 동작**: 창을 닫을 때 수행할 작업을 선택합니다 (매번 묻기, 트레이로 최소화, 종료).

### 번역 API
*   **공급자**: "Google (무료)"와 "OpenAI 호환" 간에 전환합니다.
*   **OpenAI 호환 설정**:
    *   **API URL**: 기본값은 `http://127.0.0.1:11434/v1/chat/completions` (Ollama에 완벽).
    *   **API Key**: 로컬 LLM의 경우 선택 사항입니다.
    *   **모델**: 모델 이름을 지정합니다 (예: `llama3`, `gpt-4o`, `zongwei/gemma3-translator:1b` 제가 시도해 봤는데 완벽하게 작동합니다).
